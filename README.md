# 🌧️ AP Rainfall Prediction & Early Warning System

This project develops a Machine Learning based rainfall prediction and flood risk analysis system for Andhra Pradesh.

The system predicts future rainfall using historical rainfall datasets and identifies vulnerable canals and embankments using geospatial analysis.

## 🚀 Features
- Rainfall prediction using ML models
- Village-wise and coordinate-based prediction
- Flood risk threshold alert system
- Canal and embankment impact analysis
- Interactive Streamlit dashboard
- GIS-based infrastructure risk detection

## 🧠 Technologies Used
- Python
- Machine Learning (Regression + Classification)
- Streamlit
- GeoPandas & Shapely
- Pandas & NumPy

## 📊 Data Sources
- Historical rainfall dataset (5+ years)
- Canal shapefiles
- Embankment shapefiles
- Village geographic coordinates

## 🎯 Objective
To build an early warning system supporting agriculture and water management planning in Andhra Pradesh.
